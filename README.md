# Diamond

Customer-facing digital profile project for Kushal Diamonds.

## Project Structure

- `index.html`: Single-page customer profile
- `data.json`: JSON-driven profile content
- `Assets/`: Images, icons, SVGs, and media files
- `Css/`: Legacy styles
- `Js/`: Legacy scripts

## Notes

- Data is loaded from JSON
- No admin panel is used
- Assets are mapped with relative paths for server deployment
